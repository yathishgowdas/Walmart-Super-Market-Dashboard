# 🛒 Walmart Super Market Dashboard (Power BI)

## 📌 Overview
Interactive Power BI dashboard built using Walmart sales data to analyze sales, profit, customers, and regional performance.

## 🎯 Aim
To create a professional, interactive dashboard for business insights and decision-making.

## ⚙️ Tools Used
- Power BI Desktop  
- Power Query  
- DAX  

## 🔄 Data Cleaning
- Fixed Order Date & Ship Date formats  
- Removed unnecessary columns  
- Handled missing values  
- Extracted Month & Year  

## 📊 Dashboard

### 🔹 Page 1: Overview
- KPI Cards: Sales, Profit, Quantity, States, Cities  
- Slicers: Month, Year, Category  

**Charts:**
- Sales by Category (Pie)  
- Sales by Segment (Bar)  
- Top 5 States & Cities  
- Sales by Ship Mode & Payment Mode  

### 🔹 Page 2: Details
- Top 5 Customers  
- Top 5 Products  
- Detailed Table (Sales, Profit, Location, Category, etc.)  

## 🎨 Features
- Dark professional theme  
- Fully interactive visuals  
- Dynamic filtering (e.g., Category, Month)
